# Jobjab Portfolio

A modern, responsive portfolio website built with Next.js and Tailwind CSS to showcase Web3 projects, skills, and achievements.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Icons:** Lucide React
- **Animations:** Framer Motion
